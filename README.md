##Git and Github
####Git

Git ကို ၂၀၀၅ ခုနှစ်တွင် Linus Tourvals နှင့် အခြားသော Hacker များမှ တီထွင်ခဲ့ကြပါသည်။  source control management (SCM) အဖြစ် အသစ်ထပ်မံ ထုတ်လုပ်ခဲ့ကြခြင်းဖြစ်ပါသည်။ ယခင် ရှိသော  Concurrent Versions System (CVS) နှင့်  version control system (VCS) များထက် ပိုမို အသုံးပြုရ လွယ်ကူစေရန် ရည်ရွယ်ထားပါသည်။ Linux ၏ Kernel ကို Hacker များ ဝိုင်းဝန်းပြုပြင်ရာတွင် အဓိက Tools တစ်ခုအဖြစ် စတင်အသုံးပြုခဲ့ကြပါသည်။

သင်၏ Terminal တွင် man ဖြင့်သော်၎င်း whatis ဖြင့်သော်၎င်း git အကြောင်း မေးမြန်း လေ့လာပါက အောက်ပါအတိုင်းရည်ညွှန်းထားသည်ကို မြင်တွေ့နိုင်ပေမည်။
> "git - the stupid content tracker"

GIT အခြေခံကို မြန်မာဘာသာဖြင့် လေ့လာရန် အတွက်

####Github
Git Software အထောက်အပံ့ ဖြစ်သော Github တွင် သင် အခမဲ့ အသုံးပြုနိုင်ပါသည်။ Private Repo အတွက် အခကြေးငွေ ပေးရသော်လည်း Public Repo အတွက် အခမဲ့ အသုံးပြုနိုင်ပါသည်။ Public Repo သည် လူများ ကြည့်ရှုနိုင်ခြင်း ဝင်ရောက် ကူညီ ပြင်ဆင်နိုင်ခြင်းတို့ကြောင့် ကျွန်တော်တို့၏ Wiki Project အတွပိုမို အဆင်ပြေပါသည်။
